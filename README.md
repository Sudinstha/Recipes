# Recipes

# Description
This is a simple recipe website built using only HTML.
The purpose of this project is to practice basic HTML structure,semantic elements, and listing formatting.

# Features
- Displays cooking recipes
- Uses ordered and unordered lists
- Clean and readable HTML structure
- Beginner-friendly project

# Technologies used
- HTML5
